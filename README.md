# robosys2023
講義に基づいてプログラミングを完成する

# plusコマンド
## 機能
標準入力から読み込んだ数字を足す.

## ダウンロード
wsl2をwindowsにインストールする.
* **wsl --install**

## インストール
* **git clone https://github.com/zs414/robosys2023.git**

## 実行例
* **seq 5 | ./plus**
* **15**
